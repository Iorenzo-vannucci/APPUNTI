# CyberChallenge: Integration of a Training Platform on Side-Channel HW Attacks

## `client` folder
* `challenges`
> This is a placeholder folder. Once the client environment deployed locally by the user, this folder has to be manually populated with the Jupyter Notebooks related to each Jeopardy CTF, to be downloaded as attachments from the CC.it website.
* `cyberchallenge_client`
> This folder contains our custom Python library used by the client to craft capture requests and connect to the remote server.
* `tutorials`
> This is a placeholder folder. Once the client environment deployed locally by the user, this folder has to be manually populated with the Jupyter Tutorials related to the Trainings course, to be downloaded as attachments from the CC.it website.
